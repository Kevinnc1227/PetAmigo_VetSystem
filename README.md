\## 🚀 Como executar o projeto

1\. Clone o repositório:

&#x20;  `git clone https://github.com/SEU\_USUARIO/PetAmigo\_VetSystem.git`

2\. Importe o projeto no Eclipse (File -> Import -> Existing Projects into Workspace).

3\. Execute o script SQL localizado em `/docs/script\_db.sql`.

4\. Rode a classe `Main.java`.
